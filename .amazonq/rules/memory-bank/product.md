# Product Overview

## Project Purpose
go-reloaded is a Go-based text reformatter and editor that processes input text files to produce corrected and enhanced output files. The program applies intelligent transformation rules to fix case formatting, punctuation spacing, and linguistic details while handling number conversions and article agreement.

## Key Features
- **Text Transformation Pipeline**: Applies multiple transformation stages in sequence
- **Number Base Conversion**: Converts hexadecimal and binary numbers to decimal
- **Case Transformations**: Supports uppercase, lowercase, and capitalization with multi-word support
- **Punctuation Normalization**: Proper spacing and attachment of punctuation marks
- **Quotation Mark Handling**: Removes inner spacing from quoted text
- **Article Agreement**: Automatically corrects "a" to "an" before vowels and 'h'
- **Pipeline Architecture**: Modular design for easy testing, extension, and maintenance

## Target Users
- Text processing applications requiring format standardization
- Content management systems needing automated text correction
- Educational tools for grammar and formatting assistance
- Preprocessing systems for text analysis pipelines
- Developers building text normalization workflows

## Use Cases
- Auto-editing text before publication
- Preprocessing text for further analysis
- Standardizing format consistency across documents
- Educational grammar correction tools
- Content management system text enhancement
